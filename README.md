# 🔐 Password Security Dashboard

A **cybersecurity web application** that analyzes password strength, detects potential security risks, and visualizes password attack resistance using interactive charts.

This project helps users understand how secure their passwords are using entropy calculations, breach detection, and attack simulations.

---

## 🚀 Features

- 🔑 Password Strength Analyzer
- 📊 Security Score Dashboard
- 🔍 Password Breach Detection
- ♻️ Password Reuse Detection
- 🧠 Entropy-based Security Analysis
- ⏳ Password Crack Time Estimation
- 📉 Attack Difficulty Chart (Chart.js)
- 🕸 Security Radar Visualization
- 🔐 Strong Password Generator
- 📡 Modular Cybersecurity Architecture
